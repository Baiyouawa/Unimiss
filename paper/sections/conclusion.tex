\section{Conclusion}

This paper draft presents UniMiss, a mechanism-aware imputation framework that explicitly separates foundational observed context (OO), observed interaction recovery (OM), missing-structure modeling (MM), and Stage-II MAR-vs-MNAR adaptive gating. The design is faithful to the local CIKM methodology and implemented with switchable module-level controls for rigorous ablation.

The full experiment matrix is integrated into a unified project protocol covering main comparison, hyperparameter sensitivity, visualization analysis, and lightweight/scalable evaluation. This organization keeps the code path, experimental narrative, and manuscript structure aligned around the same mechanism-aware formulation.
